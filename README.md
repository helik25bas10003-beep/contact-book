Contact Book - Python Project
Description

The Contact Book is a simple command-line Python application that allows users to manage their contacts efficiently. Users can add, view, search, and delete contacts using an easy menu-driven interface. This project is ideal for beginners to learn Python fundamentals like dictionaries, loops, and functions.

Features
Add a new contact with name and phone number
View all saved contacts
Search for a contact by name
Delete a contact
Simple and interactive menu
Installation
Make sure you have Python 3 installed.
Download or clone this repository.
Open a terminal/command prompt and navigate to the project folder.
Run the program using:
python contact_book.py
How to Use
Run the program.
Select an option from the menu:
1: Add Contact
2: View Contacts
3: Search Contact
4: Delete Contact
5: Exit
Follow the prompts to perform actions.
Contacts are stored temporarily during runtime (not saved permanently).
Technologies Used

Python Dictionaries and Functions
